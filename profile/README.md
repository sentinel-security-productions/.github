# Sentinel

### Detect threats before they become transactions.

Sentinel is an open-source blockchain security platform designed to provide real-time threat detection and monitoring across Stellar, Soroban, and EVM ecosystems.

We help developers, protocol teams, validators, DAOs, and security operators identify suspicious activity before incidents escalate into exploits, governance attacks, or treasury losses.

---

## Mission

Build the security layer blockchain ecosystems deserve.

Our goal is to make proactive threat detection accessible to every protocol, startup, and community operating on-chain.

---

## What Sentinel Does

🔍 Real-Time Transaction Monitoring

Monitor pending and confirmed blockchain activity as it happens.

🛡️ Threat Detection

Identify suspicious events including:

* Unauthorized contract upgrades
* Ownership transfers
* Governance attacks
* Large treasury withdrawals
* Abnormal wallet behavior
* High-value asset movements

🚨 Instant Alerts

Receive security notifications through:

* Discord
* Telegram

Planned integrations:

* Slack
* Email
* Webhooks

🌐 Multi-Chain Visibility

Supported ecosystems:

* Stellar
* Soroban
* Ethereum
* Polygon
* Base
* Additional EVM-compatible networks
